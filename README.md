# 🛍️ Product API

ASP.NET Core 8 ile geliştirilmiş RESTful Web API projesi.

## 🚀 Teknolojiler

- **ASP.NET Core 8** — Web API framework
- **Entity Framework Core** — ORM
- **SQL Server** — Veritabanı
- **JWT Authentication** — Token bazlı kimlik doğrulama
- **BCrypt** — Şifre hashleme
- **Swagger** — API dokümantasyonu

## 🏗️ Mimari

Katmanlı mimari (N-Tier Architecture) kullanılmıştır:

- **ProductApi** — Web API katmanı, HTTP istekleri burada karşılanır
- **ProductApi.Core** — Entity'ler ve Interface'ler
- **ProductApi.Infrastructure** — Veritabanı işlemleri, Repository'ler

## 📦 Özellikler

- ✅ Ürün yönetimi (CRUD)
- ✅ Kategori yönetimi (CRUD)
- ✅ Kullanıcı kaydı ve girişi
- ✅ JWT ile kimlik doğrulama
- ✅ Role bazlı yetkilendirme (Admin/User)
- ✅ DTO kullanımı
- ✅ Validation
- ✅ Global Exception Handling
- ✅ Repository Pattern
- ✅ Dependency Injection
